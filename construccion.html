Explicación del Proceso de Construcción

El proceso de construcción del robot incluye varios pasos fundamentales. Primero, se debe seleccionar y ensamblar la base del robot, luego conectar los motores y otros componentes electrónicos. Después, se programará el dispositivo R8+ para coordinar las funciones del robot. A continuación, te mostraremos los pasos específicos y cómo puedes trabajar con los componentes de manera eficiente.

Pasos de Construcción

Para construir el robot, sigue los siguientes pasos:

Paso 1: Armar columna y base

Primero armamos el pilar o columna donde la barrera se apoya al bajar y una base que servirá para en los siguientes pasos armar la estructura.

Paso 2: Instalar la batería y motor

En la base anteriormente creada colocar la bateria y el motor para un correcto funcionamiento de la barrera.

Paso 3: Construir la barrera

En este paso procedemos a armar la estructura de la barrera. Esta misma se ubicará en la base donde esta el motor y la batería.

Barrera terminada:

Paso 4: Armar el puente conector

Este puente es el que conectara la base con la columna y hará que la estructura se mantenga firme.

Paso 5: Incorporar actuador, sensores y bloque r8+

Conectar los dipsotivos finales que harán que la barrera funcione automáticamente.

Paso 6: Conectar cables y verificar correcto funcionamiento.

Conectar la batería y verificar que todo funcione correctamente. Luego de esto nuestra barrera de seguridad quedaría armada
